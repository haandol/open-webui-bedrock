# Open WebUI for Amazon Bedrock

This is an example of how to use [Open WebUI](https://docs.openwebui.com/) for Amazon Bedrock via [LiteLLM](https://litellm.vercel.app/docs/providers/bedrock).

# Prerequisites

- Docker Compose
- setup AWSCLI

# Usage

```bash
$ docker-compose up -d
$ open localhost:3000
```
